## <a name="demonstração"></a>💻 COMO DEMONSTRAR (PASSO A PASSO)

### **Setup da Demo (antes de mostrar)**

1. **Abra em Full Screen:**
   - URL: https://dortapods.vercel.app/
   - Coloque em tela cheia (F11)
   - Use o celular ou tablet se possível (mais impactante!)

2. **Tenha os dados de exemplo prontos**
   - 3-5 produtos cadastrados
   - Alguns com baixo estoque (para alertar)
   - Pedidos antigos para mostrar histórico

### **Sequência Recomendada**

```
1. DASHBOARD → Mostre visão geral
   "Viu? Tudo que precisa saber em uma tela"

2. PRODUTOS → Crie UM novo produto (10 segundos)
   "Viu como é rápido? Uma vez cadastrado, usa sempre"

3. ENTRADA → Registre estoque
   "Quando chega mercadoria você registra aqui"

4. PEDIDO → IMPORTANTE! Crie um pedido do zero
   a) Digite nome do cliente
   b) Clique [+] em 2-3 produtos
   c) Mostre calculando total e lucro em tempo real
   d) Clique "Finalizar"
   e) Mostre o pedido salvo no histórico
   "Viu? Estoque atualizou automaticamente!"

5. VENDAS → Mostre o pedido criado no histórico
   "Todos seus pedidos ficarão aqui para sempre"

6. VOLTE AO DASHBOARD
   "Viu como o faturamento e estoque atualizaram?"
```

### **Tips Importantes**
- 🎯 Fale DEVAGAR e deixe ele entender
- ⏱️ Não passe por tudo rápido
- 👀 Faça ele prestar atenção nos cliques [+]
- 💡 Destaque: "Tudo automático, sem contas, sem erros"
- 🎉 Quando mostrar o lucro calculado, diga:
  > "Sabe quanto você lucra em cada venda? AGORA VOCÊ SABE!"

**"Não aparece meu produto no pedido"**
→ "Seu produto tem estoque? Se tiver 0, não aparece"

**"Perdi um pedido"**
→ "Vai em VENDAS → Todos. Está lá, não apaga nunca"

**"Como faço pra resetar tudo?"**
→ "Vide arquivo LIMPAR-BANCO-MANUAL.md"

**"Posso usar em 2 celulares?"**
→ "Sim! Loga no mesmo app nos 2, sincroniza automático"
